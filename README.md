# AnimLab — CSS Animation Visualizer

[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![Status](https://img.shields.io/badge/status-active-success.svg)]()
[![Made with](https://img.shields.io/badge/made%20with-JavaScript-blue.svg)]()

AnimLab is a powerful and interactive tool to **visualize, customize, and export CSS animations** in real time. It is designed for developers, designers, and learners who want to experiment with animations without writing everything from scratch.
          
---

## Features

### Animation System
- 40+ built-in animations (Basic, Entrance, Exit, Motion, FX, Advanced V2)
- Real-time preview
- Play / Pause control
- Speed adjustment
      
### Shape Support
- Square
- Circle
- Triangle
- Text (custom input supported)

### Customization Controls
- Duration
- Delay
- Iteration count (including infinite)
- Timing functions (ease, linear, cubic-bezier, steps)
- Direction & fill mode
- Size & border radius
- Color picker (gradients + solid)

### Advanced Animations
- Liquid Wave
- Neon Pulse
- Magnetic Motion
- Elastic Spin
- Vortex
- Glitch RGB
- Warp
- Bounce Rotate
- Soft Fade Scale
- Energy Ring (SVG-based radial wave)

### Code Generation
- Auto-generated CSS
- Clean formatted output
- One-click copy to clipboard

---

## Tech Stack

- HTML5
- CSS3 (Keyframes, Gradients, Effects)
- Vanilla JavaScript (No frameworks)

---

## How to Use

1. Select an animation from the left sidebar
2. Choose a shape (square, circle, triangle, or text)
3. Adjust controls from the right panel
4. Preview animation in real time
5. Copy the generated CSS and use it in your project

---

- Animations update instantly
